# Rims

This is an Elixir + Phoenix + LiveView + Surface wrapper around [https://animate.style/](Animate.css). 

## Installation

Add to mix.exs:

```
	{:rims, git: "https://github.com/tlack/rims.git"}
```

## Usage

```
	<Rims style="lightSpeedInRight">
		<div class="h-24 bg-red text-white">
			Hello world!
		</div>
	</Rims>
```
